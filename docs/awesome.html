令人敬畏的WebSockets

WebSockets相关原理和技术的精选列表。

WebSocket 是一种计算机通信协议，通过单个TCP连接提供全双工通信信道。 WebSocket协议在2011年由IETF标准化为RFC 6455，Web IDL中的WebSocket API由W3C标准化。

每种语言的工具

Agnostic

  • Websocketd - UNIX方式的WebSockets - Web浏览器和服务器之间的全双工消息传递。

  • Nchan - Nchan是一个可扩展，灵活的现代Web发布/订阅服务器，构建为Nginx Web服务器的模块。

  • Apache-websocket - Apache WebSocket模块。

  • MinnowServer - 超小型快速嵌入式HTTP（S）WebSocket服务器。

  • gwsocket - 快速，独立，与语言无关的WebSocket服务器RFC6455兼容。

C

  • Libwebsockets - 它是一个轻量级的纯C库，可以使用最少的CPU和内存资源，并在客户端或服务器的两个方向上提供快速吞吐量。

  • Libwebsock - 用于简易WebSockets服务器的C库。

  • Websocket - 用C编写的Websocket服务器

  • facil.io - 用于Web应用程序的服务器/框架库，包括Websockets和本机 pub/sub 。

  • libuwsc - 基于libubox for Embedded Linux的轻量级和完全异步的WebSocket客户端C库。

  • mongoose - Mongoose嵌入式Web服务器库 - Mongoose不仅仅是一个嵌入式网络服务器。它是一个多协议嵌入式网络库，具有TCP，HTTP客户端和服务器，WebSocket客户端和服务器，MQTT客户端和代理等功能。

C++

  • Websocketpp - C ++ Websocket客户端/服务器库。

  • QtWebSockets - QtWebSockets模块是Qt5库的附加组件。

  • Beast - HTTP和WebSocket构建于C ++ 11中的Boost.Asio上。

  • µWebSockets - 高度可扩展的WebSocket服务器和客户端库。

  • 简单的WebSocket，服务器 - 一个非常简单，快速，多线程，独立于平台的WebSocket（WS）和WebSocket安全（WSS）服务器和客户端库，使用C ++ 11，Boost.Asio和OpenSSL实现。

  • UEWebsocket - 适用于c ++和蓝图开发人员的虚幻引擎4 websocket插件。

  • IXWebSocket - 支持TLS的轻量级C ++ 11多线程客户端库。

C#

  • ASP.NET SignalR - 令人难以置信的简单的.NET实时网络。

  • WebSocketListener - 用于.NET / Mono的轻量级且高度可扩展的异步WebSocket服务器。

  • websocket-rpc - 用于.NET的WebSocket RPC库，具有自动JavaScript客户端代码生成功能，支持ASP.NET Core。

  • NetGain - 一个支持Stack Overflow的高性能websocket服务器库。

  • websocket-manager - ASP .NET Core的实时库。

  • WebSockets - WebSocket协议的实现，以及客户端和服务器集成组件。

  • WebSockets support in ASP.NET Core - 本文介绍如何在ASP.NET Core中开始使用WebSockets。

Erlang

  • Sockjs-erlang - WebSocket仿真 - Erlang服务器。

  • Cowboy - 用Erlang编写的小型，快速，模块化的HTTP服务器。

  • Kraken - 用于实时应用程序的分布式Pubsub服务器。

Go

  • Gorilla Websocket - Go的WebSocket实现。

  • Websocket - Package Websocket实现了RFC 6455中指定的WebSocket协议的客户端和服务器。

  • Ws - Go的Tiny WebSocket库。

  • 1m-go-websockets - 在Go中处理1M websockets连接。

  • gotify/server - 一个简单的服务器，用于每个Web套接字实时发送和接收消息。

  • nhooyr/websocket - Go的最小和惯用的WebSocket库。

Haskell

  • Websockets - 用于创建支持WebSocket的服务器的Haskell库。

Java VM

  • Chord - 库旨在弥合CLJ / CLJS三元组，web-socket和core.async之间的差距。

  • Luminusweb - Luminus是一个基于一组轻量级库的Clojure微框架。

Java

  • Java-WebSocket - Barebones WebSocket客户端和服务器实现用100％Java编写。

  • Atmosphere - 用于JVM的实时客户端服务器框架，支持具有跨浏览器回退的WebSocket。

  • Webbit - 基于Java事件的WebSocket和HTTP服务器。

Kotlin

  • Scarlet - Tinder的Retrofit启发了Kotlin，Java和Android的WebSocket客户端

Scala

  • Play - Java和Scala的高速Web框架。

  • Finagle-websocket - Finagle Websocket客户端和服务器。

Node.js

  • Socket.IO - 拥有最快，最可靠的实时引擎。

  • Nodejs-websocket - 用于websocket服务器和客户端的Node.js模块。

  • WebSocket-Node - Node.JS的WebSocket实现（草案-08到最终的RFC 6455）。

  • Sockjs-node - WebSocket仿真 - Node.js服务器。

  • Ws - ws: Node.js最快的跨平台RFC-6455 WebSocket实现。

  • deepstream.io - 为实时移动，Web和iot提供快速，安全，可扩展的实时服务器。

  • websocket-as-promised - 基于Promise的W3C WebSocket包装器：允许在与WebSocket服务器连接，断开连接和消息传递时使用promises。

  • faye-websocket-node - 符合标准的WebSocket客户端和服务器。

  • ws-wrapper - 轻量级WebSocket包装器，提供类似socket.io的事件处理程序API以及基于Promise的请求。

  • ws-server-wrapper - 用于服务器端的ws-wrapper的配套库。

  • uws - Tiny WebSockets（通过Node.js访问C ++库，μWebSockets）

  • netflux - 基于WebRTC和WebSocket的JavaScript客户端和服务器端传输API

  • Sockette - WebSocket客户端，如果连接丢失，将自动重新连接。

Perl

  • Net::WebSocket::Server - 简单易懂的Perl WebSocket服务器，具有最小的依赖性。

PHP

  • Ratchet - Ratchet是一个松散耦合的PHP库，为开发人员提供了通过WebSockets在客户端和服务器之间创建实时双向应用程序的工具。

  • Phpws - PHP Web Socket服务器。

  • Sandstone - Microframework构建实时Rest API。

  • Ws4py - 适用于Python的WebSocket包。

  • Tornado - Tornado是一个Python Web框架和异步网络库，最初是在FriendFeed上开发的。

  • Websockets - Python 3中的WebSocket实现。

  • Em-websocket - 基于EventMachine的WebSocket服务器。

  • Faye-websocket-ruby - 符合标准的WebSocket客户端和服务器。

  • Iodine - WebSocket / HTTP服务器，集成了pub / sub和可选的Redis支持。

  • Websocket-driver-ruby - 具有可插入I / O的WebSocket协议处理程序。

  • Websocket-ruby - 通用Ruby库来处理WebSocket协议。

  • Scorched - Ruby的轻量级Web框架。

  • Firehose - 构建实时Ruby Web应用程序。由Poll Everywhere的优秀人员创建。

  • render_sync - 实时Rails部分。

  • websocket-rails - 即插即用的websocket支持轨道上的红宝石。
